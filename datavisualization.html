Data Visualization Project

This project was part of my Arts Application Programming course in the B.S. Degree of Creative Computing at Queen Mary University of Lodon, 2021.

In this project we were required to analyse a database and identify the important trends.
Then we had to come up with a design and finally to program an interactive data visualization in Processing (Java based) programming Language.
